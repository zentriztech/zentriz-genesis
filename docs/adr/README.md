# ADR — Architecture Decision Records

## Propósito

ADRs documentam **decisões arquiteturais** importantes do projeto, preservando o **contexto**, **alternativas consideradas** e **consequências** para futuros desenvolvedores e chats.

## Formato

Cada ADR segue o template em [0000-template.md](0000-template.md).

## Convenções

- **Status**: Proposto | Aceito | Deprecado | Substituído por ADR-XXXX
- **Data**: YYYY-MM-DD
- **Contexto**: Por que a decisão foi necessária
- **Decisão**: O que foi decidido
- **Consequências**: Impactos e trade-offs
