# Checklist Web (React)

## Arquitetura
- [ ] Estrutura de pastas (features/components/pages)
- [ ] State management definido (Context/Redux/Zustand/React Query)
- [ ] Rotas definidas e protegidas (se necessário)
- [ ] Erros e loading states padronizados

## Qualidade
- [ ] ESLint + Prettier configurados
- [ ] Tests (unit + integração; e2e se aplicável)
- [ ] Build produz artefatos reprodutíveis

## Performance
- [ ] Lazy loading onde fizer sentido
- [ ] Evitar re-render excessivo

## Segurança
- [ ] Sanitização de input (quando aplicável)
- [ ] Sem secrets no bundle
